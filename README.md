#集群高可用架构整理(不断更新中)
>在大型系统或是对稳定性要求较高的项目中，集群高可用架构是必不可少的，这里整理一些平时用到过的集群/高可用技术，慢慢更新中。。。

####二种最基本的集群模型：

-  M-S 型集群: M-S 形式的集群一般用于高可用架构，主节点挂掉时从节点顶上，一般不具有分流 (负载均衡) 的作用。  
-  M-M 型集群: M-M 形式的集群一般用于分流（负载均衡），部分 M-M 形集群会包含 M-S 型的主从功能 (像 Redis Cluster )，分流的同时达到高可用的作用。
